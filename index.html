Features

Everything you need to manage your Claude Code workflow

Auto-Detection & Live Status

Automatically discovers all Claude Code sessions from local files and shows their real-time state: running, tool executing, waiting for input, idle, completed. Search, filter, and group in one sidebar.

Session Resume

Click any session to instantly resume with claude --resume. Rename, archive, and organize your sessions. Lazy-loaded terminals — only created when you click.

Multi-Terminal Tabs

Multiple terminals per project — Claude + shell tabs side by side. Tabs persist across restarts. Configurable presets (Standard, iTerm2, Minimal).

Git Integration

Live git status with diff viewer (click any file), worktree modal, commit & PR in embedded shell tabs. Branch display in the header.

Quick Actions

Commit, create PR, worktree, open in IDE, Finder, or terminal. Fully customizable — reorder, show/hide, choose your external terminal (iTerm2, Warp, Alacritty).

Real Usage Tracking

Live data from Claude's /usage command: session %, week %, extra spending, reset countdown. Rate limit detection. Configurable refresh interval.

Fully Configurable

Dark/Light theme, EN/FR, terminal presets, layout, IDE detection, notifications, tray icon, demo mode, session sorting — all from Settings (Cmd+,).

Native macOS & Secure

Signed & notarized by Apple, auto-updates via GitHub Releases. Tray icon, native notifications, macOS menus. Context isolation, execFile only, no telemetry. Everything runs locally.

Auto-detects installed IDEs and adds them as quick actions. One click to open any project.
